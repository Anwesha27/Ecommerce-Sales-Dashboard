# Madhav Ecommerce Sales Dashboard

1. An interactive Power BI dashboard designed to analyze ecommerce sales performance, profitability, customer behavior, and payment trends across different regions and time periods.

2.This dashboard provides a comprehensive overview of ecommerce sales data, enabling users to track revenue, profit, quantity sold, and average order value (AOV).
It helps identify high-performing states, categories, sub-categories, customers, and seasonal profit trends to support data-driven business decisions.

8. Key Insights
Maharashtra contributes the highest sales revenue among all states
Clothing category dominates overall quantity sold
Printers and Bookcases generate the highest profit among sub-categories
COD is the most frequently used payment mode
Profitability shows noticeable seasonal variation across months
